The Box-Model

Every element in a page is a box and it is the property of the box that let's us build complex web-pages with different alignments. Here we will look at different Box-properties.

Display Types

Box has two display types : inner and outer display type. Each of those property can take value which sets the behavious of the boxes which sets the flow of the page.

Box-Model

Box-Model in CSS includes working with different properties of the box like height, width, padding, margin which entirely builds a block on the page. Based on how the total height & width of the box is calculated the Box-Model is categorised into two types .

  • Standard Box-Model (default)
  • Alternate Box-Model

Margin-Collapsing

Margins of two adjacent blocks are not added to get the final margin, the largest value between them is selected. If the margin is negative then the space between the boxes decreases and might overlap with other element.
