Pseudo Selector

We can get HTML elements of the page in different ways, one such way is using the Pseudo-selectors.

There are two types of pseudo-selectors :

  • Pseudo Class
  • Pseudo Elements

Pseudo Class :

This selectors are used to select the elements based on the state of the element. After selecting the element it adds the styling property to those elements, behavior is like adding a class to the elements but using pseudo-class makes the code more maintainable.
