Pseudo Selectors

There are two pseudo-selectors that helps us to grab the HTML elements in a different way.

  • Pseudo-Class : Selects an element based on its state (hover, active).
  • Pseudo-Element : Selects a portion of specified Element based on the given property.

Pseudo-Class : This will change it's color on hovering-up!!

Pseudo Element : Only first letter of the paragraph will be styled. Rest of the line will be normal.

Multiple Classes

Some Element might have multiple classes and so we may want to style the element if it has specific set of classes else not to apply the style. It is done by mentioning classes with no space.

Example : Having all the 3 classes

<div class="blackBorder blueFont italics">This has all the classes</div>
This has all the classes

Example : Having only the 2 classes

<div class="blackBorder italics">This has all the classes</div>
This does not have all the classes
