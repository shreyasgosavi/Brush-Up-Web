Cascade Layers

We can create multiple layers of style in the same origin which helps to organise the styling in structured way and also avoids the problem of specifity as the new layers has precedance over the previous layers.

Defining Layers

There are 3 ways of defing the layers in CSS.

  • @layer statement at-rule, where we declare the layer names. It has no style body.
  • @layer block at-rule, styles are defined in the block.
  • @import rule with the layer keyword which can assign external sheets as a CSS layer.
