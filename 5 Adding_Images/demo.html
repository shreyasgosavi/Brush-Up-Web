Flow of the document

A document has several sections which can be identified by HTML-tags. Some are used for providing several properties to the document and some acts as a container for the content of the document.

Lets look at the main HTML tags :

!DOCTYPE :
Provides information to Browser on how to render the document.
HTML :
It is where the content of the page resides which includes the head and body elements.
Head :
It contains meta-data of the document which includes :
  • Character-Encoding of the document
  • Document Title
  • Document Description
  • Author Name
  • Links to several resources that can be embeded to the page
Body
It consists of the contents that is visible in the browser which has several other elements that makes it possible to structure the page in a way we want.

Order Of Elements :

Elements listed above should be placed in the same order !!
